НОВАЯ КОМАНДА

Операция WDTT

Для состояния укажите true или false; для срока — ISO‑дату или null.

Команда уйдёт только выбранному активному узлу. Повторные отправки защищены ключом идемпотентности.

СТАТУСЫ

Как это работает

  1. Оператор создаёт ограниченную команду.
  2. Агент забирает только свои команды.
  3. Агент подтверждает результат без логов и секретов.

Создание и изменение используют только поля контракта WDTT.
